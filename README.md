## Getting Started


### Installation

<p>1. Clone the repository</p>

```
git clone https://github.com/IlyashenkoA/magebit-project.git
```

<p>2. Install all packages</p>

```
npm install
```

<p>3. Run the project</p>

```
npm start
```
